MMKPC Content Gate Privacy Policy

MMKPC Content Gate is a browser extension by Matthew E. Mitchell / MMKPC Studios that blocks selected websites until the user has posted content for the day or starts a short study session.

Website: https://github.com/MMKPC

Developer: Matthew E. Mitchell, MMKPC Studios

Data Stored

MMKPC Content Gate stores the following information in Chrome extension storage:

  • Blocked site domains configured by the user.
  • Daily content idea, publish target, posted status, and proof URL.
  • Temporary study-session reason, start time, and expiration time.

Data Sharing

MMKPC Content Gate does not sell, transfer, or share user data with third parties.

Network Use

MMKPC Content Gate does not send user data to an external server. Proof URLs are stored only so the extension can decide whether the daily gate should open.

User Control

Users can edit blocked sites from the extension options page and can remove all stored extension data by uninstalling the extension or clearing extension storage in Chrome.
